## デプロイ手順

### Step 1: 事前確認

```bash
cd companies/npb-predictions/web

# Cloudflare ログイン確認
wrangler whoami

# ビルド確認
npm run build:cf

# ローカル動作確認 (オプション)
npx wrangler pages dev .vercel/output/static --d1=DB
```

### Step 2: D1 本番データベース初期化

```bash
# 本番D1にシードデータ投入
wrangler d1 execute npb-predictions --file=src/db/seed-commentators.sql

# 投入確認
wrangler d1 execute npb-predictions \
  --command="SELECT COUNT(*) as users FROM users"
# 期待値: users=29, predictions=29, ranking_picks=348, title_picks=60
```

### Step 3: Cloudflare Pages デプロイ

```bash
# 本番デプロイ
npm run deploy:cf

# または直接
wrangler pages deploy .vercel/output/static \
  --project-name=npb-predictions \
  --branch=main
```

### Step 4: デプロイ後確認
