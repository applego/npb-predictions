## 1. SEO戦略

### 1.1 キーワード設計

3層のキーワード戦略で、検索ボリュームの異なるユーザーを段階的に獲得する。

#### Tier 1: ロングテール（低競合・高CVR）--- 即効性あり

これらは大手スポーツメディアが手薄な領域。既存の解説者データ1,295人分が最大の武器。

| キーワード | 月間推定検索数 | 対応ページ | 優先度 |
|-----------|-------------|-----------|-------|
| `[解説者名] 順位予想` (例: 鳥谷敬 順位予想) | 100-500/人 | `/rankings/commentators/[slug]` | 最高 |
| `[解説者名] 的中率` | 50-200/人 | 同上 | 最高 |
| `[年度] セリーグ 順位予想 解説者` | 500-2,000 | `/seo/[year]/commentator-accuracy` | 高 |
| `[年度] パリーグ 順位予想 結果` | 300-1,000 | `/seo/[year]/[league]/final-standings` | 高 |
| `プロ野球 順位予想 当たった人` | 500-1,500 | `/rankings/all-time` | 高 |
| `[チーム名] [年度] 順位予想` (例: 阪神 2026 順位予想) | 200-800/チーム | `/seo/[year]/teams/[team]` | 中 |

#### Tier 2: ミドルテール（中競合）--- 3-6ヶ月で狙う

| キーワード | 月間推定検索数 | 対応ページ |
|-----------|-------------|-----------|
| `プロ野球 順位予想 2026` | 10,000-30,000 | `/seo/2026` |
| `セリーグ 順位予想` | 5,000-15,000 | `/seo/2026/central/final-standings` |
| `パリーグ 順位予想` | 3,000-10,000 | `/seo/2026/pacific/final-standings` |
| `プロ野球 解説者 ランキング` | 1,000-3,000 | `/rankings/commentators` |
| `プロ野球 順位予想 的中率` | 500-2,000 | `/rankings/all-time` |

#### Tier 3: ビッグワード（高競合）--- 長期目標

| キーワード | 月間推定検索数 | 戦略 |
|-----------|-------------|------|
| `プロ野球 順位予想` | 50,000-100,000 | Tier 1-2で被リンク・権威性を積み上げてから狙う |
| `NPB 予想` | 10,000-30,000 | サイト全体のドメイン評価向上で対応 |

### 1.2 SEOページ拡充計画

現在37ページが公開済み。以下を追加する。

#### 即座に作成すべきページ（データは既にDBに存在）

1. **解説者個別ページの充実** --- `/rankings/commentators/[slug]`
   - 現在156人分存在。1,295人分に拡大する（全解説者のページ生成）
   - 各ページに「この解説者の年度別予想一覧」「的中率推移グラフ」「予想傾向分析」を追加
   - これだけで **1,000ページ以上** のSEO対象ページが生まれる

2. **年度 x 解説者 クロスページ** --- `/seo/[year]/commentators/[slug]`
   - 「鳥谷敬 2025 順位予想」のような検索に対応
   - 13年分 x 主要解説者 = 大量のロングテールページ

3. **チーム x 年度 予想者一覧** --- `/seo/[year]/teams/[team]/predictions`
   - 「巨人 2026 順位予想 誰が」のような検索に対応
   - 各チームを何位に予想した解説者の一覧

4. **「予想が当たる解説者」特集ページ** --- `/features/most-accurate`
   - 通算的中率トップ10の特集
   - 各リーグ別の最強予想家

#### シーズン連動型ページ（自動更新）

5. **今週の的中速報** --- `/news` (既存だが強化)
   - 「先週の順位変動で誰の予想が近づいた/離れた」を自動生成
   - 毎週月曜に自動更新 → Google に新鮮なコンテンツと認識させる

6. **月間レポート** --- `/reports/[year]/[month]`
   - 月末に自動生成される「今月のMVP予想家」「的中率変動」

### 1.3 テクニカルSEO改善

| 項目 | 現状 | 改善案 |
|------|------|--------|
| sitemap.xml | 基本ページ + 年度ページ | 解説者全ページ + チームページを追加 |
| robots.txt | `/admin` `/api/` を除外 | 現状OK |
| OGP画像 | `/api/og/[type]` で動的生成 | 解説者個別のOGP画像を追加（顔写真がなくても名前+スコア） |
| 構造化データ | BreadcrumbJsonLd + CollectionPage | FAQ schema, Person schema（解説者）, Dataset schema を追加 |
| Core Web Vitals | edge runtime | 現状良好。広告導入時にCLS注意 |
| 内部リンク | InternalLinksコンポーネント | 解説者ページ間の相互リンク強化 |
| canonical URL | 設定済み | `npb-predictions.pages.dev` に統一（vercel.app参照が一部残存、要修正） |

### 1.4 被リンク獲得戦略

1. **データ引用される存在になる**
   - 「解説者の順位予想的中率」は他にデータベース化しているサイトがほぼない
   - 野球ブログ・まとめサイトが引用元として自然にリンクする可能性が高い
   - 各ページに「このデータを引用する場合はリンクをお願いします」の文言を追加

2. **はてなブックマーク / note での記事公開**
   - 「13年分のデータで判明！順位予想が最も当たる解説者ランキング」
   - note記事 → サイトへの導線

3. **Qiita / Zenn での技術記事**
   - 「1,295人の順位予想データをNext.js + D1で可視化した話」
   - 開発者コミュニティからの被リンク + エンジニア層の認知

## 4. 口コミ施策

### 4.1 シェア導線の最適化

既存の ShareButton コンポーネント（X, LINE, URLコピー対応済み）を強化する。

#### シェアトリガーの追加

| トリガー | タイミング | シェア文言 |
|---------|----------|----------|
| 予想登録完了 | 予想確定時 | 「2026年セ・リーグ優勝は[チーム名]と予想！あなたはどこ？ #NPB予想リーグ」 |
| スコア変動 | 週次スコア更新時 | 「今週 +8点で〇位にランクアップ！ #NPB予想リーグ」 |
| 的中 | 予想が的中した時 | 「[チーム名]の順位ピタリ的中！+5点GET #NPB予想リーグ」 |
| ランキング変動 | 順位が上がった時 | 「予想ランキング〇位に浮上！ #NPB予想リーグ」 |
| 月間王者 | 月間MVP発表時 | 「[ユーザー名]が月間王者に！ #NPB予想リーグ」 |

#### OGP画像の強化

- 予想登録完了時に個人用OGP画像を自動生成（`/api/og/prediction`）
- 画像に「チームロゴ風カラー」+「予想順位」を表示
- X/LINEでシェアした時にリッチなプレビューが出る → クリック率UP

### 4.2 ユーザー参加型コンテンツ

#### 「あなた vs 解説者」スコア比較

- 自分の予想スコアと、全解説者の中で何位相当かを表示
- 「156人中〇〇位！[解説者名]より上！」 → シェアしたくなる
- `/users/[userId]` に解説者との比較セクションを追加

#### 予想カード（画像）の自動生成

- 予想登録時に「予想カード」画像を自動生成
- セ・パ両リーグの予想順位をカードデザインで表示
- ダウンロード or 直接シェア可能
- Instagram ストーリーズ向けの縦型フォーマットも対応

## 5. 対決グループ機能のマーケティング活用

### 5.1 機能概要

友達を招待 → グループ内ランキングで順位予想の的中率を競う機能。SPEC.md の Phase 4 に該当。

### 5.2 マーケティング活用のポイント

#### バイラルループの設計

```
[ユーザーA] がグループ作成
    ↓ 招待リンクをX/LINEでシェア
[友人B, C, D] がリンクから参加 → アカウント登録
    ↓ 各自が予想を登録
[全員] がグループ内ランキングを確認
    ↓ 「現在1位！」をXでシェア
[友人B の知人E, F] が興味を持ちサイト訪問
    ↓ 新グループ or 既存グループに参加
... (拡散ループ)
```

#### 招待インセンティブ

| レベル | 招待人数 | 特典 |
|-------|---------|------|
| ブロンズ | 3人招待 | グループ名のカスタマイズ |
| シルバー | 5人招待 | 限定バッジ（プロフィール表示） |
| ゴールド | 10人招待 | 「解説者データ詳細分析」機能の早期アクセス |

#### グループ活用シーン

| シーン | ターゲット | マーケティング施策 |
|-------|----------|----------------|
| 会社の同僚同士 | 30-50代男性 | 「社内野球リーグ」テンプレート提供 |
| 大学野球サークル | 18-22歳 | 学生向けキャンペーン（春の新歓時期） |
| 野球バー/スポーツバー | 常連客 | 店舗向け QR コードポスター |
| 親子 | ファミリー層 | 「親子予想対決」テンプレート |
| YouTube コミュニティ | 野球YouTuber の視聴者 | YouTuber とのコラボ企画 |

### 5.3 グループ機能リリース時のマーケ施策

#### Phase 1: クローズドベータ（リリース前）

1. 既存ユーザーから10グループをテスト招待
2. フィードバック収集 + 改善
3. 「ベータテスター限定バッジ」付与

#### Phase 2: ローンチ（シーズン開幕に合わせる）

1. **X キャンペーン**: 「グループ作成 + 3人招待で〇〇プレゼント」
2. **YouTuber コラボ**: 野球系YouTuber に「視聴者 vs YouTuber 予想対決グループ」を作ってもらう
3. **LINEオープンチャット連携**: 野球系LINEオープンチャットで招待リンクをシェア

#### Phase 3: グロース（シーズン中）

1. **週間グループランキング**: 最もアクティブなグループを表彰
2. **グループ間対決**: グループ同士の対決機能（平均スコアで競う）
3. **企業対抗戦**: 企業名でグループ登録 → 「企業対抗 NPB 予想リーグ」

### 5.4 シェア文言テンプレート（グループ招待時）

```
招待メッセージ（X向け）:
「プロ野球順位予想で勝負しよう！
グループ「[グループ名]」に招待されました。
解説者1,295人のデータと比較して、あなたの予想力を試そう。
→ [招待リンク] #NPB予想リーグ」

招待メッセージ（LINE向け）:
「プロ野球の順位予想で勝負しない？
[招待者名]が「[グループ名]」を作ったよ。
解説者より当てられるか挑戦してみて！
→ [招待リンク]」
```
