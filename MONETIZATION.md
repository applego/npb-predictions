# NPB Predictions League — 収益化戦略

## 概要

NPB Predictions League の収益モデルは4つの柱で構成する。
**Phase 2 以降**で段階的に実装する。Phase 1 (MVP) は無収益で信頼醸成を優先。

---

## 収益柱一覧

| # | 柱 | フェーズ | 月間目標 (成熟期) |
|---|---|---|---|
| 1 | Google AdSense | Phase 2 | ¥3,000〜¥15,000 |
| 2 | アフィリエイト | Phase 2 | ¥5,000〜¥30,000 |
| 3 | YouTube 広告収益 | Phase 3 | ¥2,000〜¥20,000 |
| 4 | スポンサー | Phase 4 | ¥20,000〜¥100,000+ |

---

## 1. Google AdSense（SEO ページ）

### 概要

SEO 流入ページ（過去順位・選手成績・予想振り返り）に AdSense 広告を配置。
PV が稼げるページを中心に収益化。

### 実装要件

- **前提**: SEO ページ 10 本以上公開 (hw-d7sat 完了後)
- **コード変更**: `app/layout.tsx` に `<Script>` タグ追加（AdSense js）
- **広告枠配置候補**:
  - 記事上部 (320x100 or レスポンシブ)
  - 記事中段 (300x250)
  - サイドバー (Desktopのみ)
- **除外ページ**: `/predict`（入力画面）、`/scoreboard`（毎日見るコア画面）

### ターゲット KPI

| KPI | 目標 |
|---|---|
| 月間 SEO PV | 10,000 PV |
| CTR | 1〜3% |
| CPC 平均 | ¥15〜¥50 |

### アクションアイテム

- [ ] AdSense 申請（SEO ページ 10 本公開後）
- [ ] `next.config.ts` に AdSense Publisher ID を env 経由で設定
- [ ] SEO ページレイアウトに広告コンポーネント追加
- [ ] Core Web Vitals を維持（CLS スコア 0.1 未満を死守）

---

## 2. アフィリエイト

### 概要

野球ファンの購買行動に沿ったリンクを設置。信頼性のある流れで自然にクリックを誘導する。

### アフィリエイト先と実装場所

| プログラム | 対象商品 | 配置場所 |
|---|---|---|
| **Amazon アソシエイト** | グローブ・バット・ユニフォーム・スコアブック | SEO ページ sidebar, チーム紹介ページ |
| **楽天アフィリエイト** | チケット・野球用品・書籍 | 試合日程ページ |
| **チケットぴあ / ローチケ** | NPB 公式チケット | 試合日程・スコアボードページ |
| **DAZN アフィリエイト** | 年間視聴パス | ページ下部 CTA バナー |
| **スポナビライブ / ABEMAプレミアム** | 視聴権 | 同上 |

### DAZN アフィリエイト詳細

- **登録先**: A8.net または バリューコマース
- **成果単価**: 契約1件 ¥1,500〜¥3,000
- **配置**: ヘッダーバナー（シーズン開幕前後に強化）

### 実装要件

- `lib/affiliate.ts`: アフィリエイトリンク生成ユーティリティ
- `components/AffiliateCard.tsx`: 商品カード UI
- `components/AffiliateBanner.tsx`: バナー UI（DAZN / チケット用）
- 全リンクに `rel="nofollow sponsored"` を付与
- クリック計測: PostHog カスタムイベント `affiliate_click`

### アクションアイテム

- [ ] Amazon アソシエイト申請（サイト公開後 90 日以内に 3 クリック必要）
- [ ] A8.net でアフィリエイト会員登録
- [ ] DAZN パートナー申請
- [ ] チケット系アフィリエイトリンク取得

## 3. YouTube 広告収益

### 概要

`npb-predictions-ja` チャンネル（hw-rh0n1）でシーズン中の日次 Shorts + 週次長尺を配信。
YouTube Partner Program (YPP) 達成後に広告収益化。

### YPP 達成条件

| 条件 | 目標 |
|---|---|
| チャンネル登録者 | 500 人（Shorts 収益は 1000 人） |
| 直近 90 日の視聴時間 | 3,000 時間 (長尺) or Shorts 300 万回 |

### コンテンツ計画（サイト連動型）

| フォーマット | 頻度 | 内容 | CTA |
|---|---|---|---|
| Daily Shorts (60 秒) | 毎日（シーズン中） | 前日結果 + 今日の予想ポイント | サイトで予想を確認 URL |
| 週次長尺 (8〜15 分) | 週 1 本 | 週間スコア振り返り + 来週の見どころ | 5 人の予想比較リンク |
| 開幕 / 節目特別 | 月 1〜2 本 | 順位予想発表 / 中間総括 | 予想登録リンク |

### サイト連動の仕組み

- 動画説明欄に `https://npb-league.com` の固定リンク
- 動画内エンドカード → サイト URL
- SEO ページ上部に YouTube 埋め込みプレーヤー（対象シーズンの最新動画）

### アクションアイテム

- [ ] `hohho-youtube-studio/channels/npb-predictions-ja/config.yaml` 作成 (hw-rh0n1)
- [ ] サイト上に YouTube 埋め込みコンポーネント追加 (Phase 3)
- [ ] 概要欄テンプレートにサイト URL を固定

## 4. スポンサー

### 概要

Phase 4 から。月間 PV 50,000 超・YouTube 登録 5,000 人超を達成してから交渉開始。

### ターゲット企業

| カテゴリ | 候補 | アプローチ |
|---|---|---|
| **野球用品メーカー** | ミズノ, ゼット, SSK, ローリングス | PR 商品提供 or バナー広告 |
| **スポーツバー / 居酒屋** | 野球観戦特化の飲食店 | ロゴ掲載 + イベント告知 |
| **チケット二次流通** | チケット流通センター, StubHub Japan | バナー + CPC 課金 |
| **スポーツメディア** | Full-Count, パーソル CSL | 記事コラボ |
| **野球教室 / スクール** | 地域少年野球団体 | ローカル広告 |

### 料金体系（参考）

| 枠 | 掲載位置 | 月額目安 |
|---|---|---|
| ゴールド | ヘッダーロゴ + 全ページ footer | ¥50,000〜 |
| シルバー | サイドバー固定バナー (300x250) | ¥20,000〜 |
| ブロンズ | SEO ページ記事内バナー | ¥5,000〜 |

### 実装要件

- `components/SponsorBanner.tsx`
- `app/api/admin/sponsors/route.ts`（スポンサー登録 API）
- スポンサーページ: `/sponsors`（メディアキット掲載）

```

```
